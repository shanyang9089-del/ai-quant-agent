# First Strategy Task

请基于 Strategy Prompt Master 的统一要求，再额外满足以下条件，为我生成一个第一版候选策略：

- 策略方向：趋势跟随
- 时间周期：15m
- 交易方向：仅做多
- 核心思路：在明确上涨趋势中寻找回踩后的继续上涨机会
- 趋势判断：使用 EMA 快线、EMA 慢线、EMA 长期趋势线
- 动量确认：使用 RSI
- 过滤条件：使用成交量过滤，避免假突破
- 入场要求：价格在长期趋势线上方，快线高于慢线，RSI 不低于中性值，成交量高于均量
- 出场要求：趋势减弱时离场，或触发固定止损/止盈
- 止损：2% 到 3% 之间，写成可配置参数
- 止盈：4% 到 6% 之间，写成可配置参数
- 要求代码尽量简洁，适合后续做参数优化和自动进化

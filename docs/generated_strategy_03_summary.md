# RSIMultiMAVolBetV1 策略说明

## 1. 策略名称
RSIMultiMAVolBetV1

## 2. 策略类型
低过滤、偏“反转起点下注”的做多原型（非稳健趋势跟随系统）。

## 3. 核心逻辑
该策略运行在 15m 周期，面向 futures 场景，`can_short = False`，通过 `leverage()` 固定使用 20x（受 `max_leverage` 限制）。

核心不是追涨，而是尝试捕捉“从弱转强”的起点：
- 价格刚重新站回快线（EMA8）
- RSI 刚从偏弱区上穿阈值（45）
- 同时成交量高于均量

只加了极少量结构限制（EMA21 >= EMA55）与轻量“不过度追高”约束（close 不高于 EMA21 的 1.02 倍），避免完全无脑接刀或追高。

## 4. 入场条件
满足以下条件即触发 `enter_long`：
1. `close > ema_fast` 且上一根 `close <= ema_fast`（重新站上快线）
2. `rsi > 45` 且上一根 `rsi <= 45`（从偏弱区上穿）
3. `volume > volume_ma` 且 `volume > 0`（量能确认）
4. `ema_mid >= ema_slow`（轻量结构过滤）
5. `close <= ema_mid * 1.02`（避免明显追高）

## 5. 出场条件
- 主出场机制：`minimal_roi = {"0": 0.05}`，即固定约 5% 止盈（1:2 风格）。
- `stoploss = -0.99` 仅作为系统兼容兜底，不依赖复杂止损逻辑。
- `populate_exit_trend()` 不使用复杂技术信号，保持极简。

## 6. 风险点
1. 这是低过滤策略，假突破与“站上即回落”会较多。
2. 在高杠杆（20x）下，即使小仓位，净值波动依然可能很大。
3. `stoploss` 设得很宽，实际风险更多依赖仓位控制与交易频率控制。
4. 策略并不追求高胜率，存在较强运气成分。

## 7. 最适合的行情
- 短线下探后快速修复的反弹段。
- 有一定流动性、量能能配合放大的币对。
- 震荡偏强、并非单边衰退的阶段。

## 8. 最容易失效的行情
- 单边阴跌/持续弱趋势（会频繁触发“假反转”）。
- 无量震荡（量能条件失真）。
- 突发消息导致的剧烈反向波动。

## 9. 为什么这是“低过滤、偏赌点”原型
1. 指标仅 3 类：RSI、EMA、成交量均线。
2. 入场条件数量少，参数少，避免堆叠复杂过滤。
3. 不构建复杂风控和复杂退出信号。
4. 明确以“反转起点下注”为定位，接受较高不确定性，强调可运行与可回测的基线属性。

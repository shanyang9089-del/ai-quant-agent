# RSIMultiMAVolBetV1 策略说明（覆盖式更新版）

## 1. 策略名称
RSIMultiMAVolBetV1

## 2. 策略类型
简单、低过滤的趋势跟随（15m / futures / 仅做多）

## 3. 核心逻辑
该策略不再做“反转第一脚”猜测，而是直接做顺势跟随：
- 当多均线结构明确向上时，只做多；
- 用 RSI 确认动量仍在多头侧；
- 用成交量高于均量做最基础的有效性确认。

整体目标是：少条件、可解释、不过度工程化，便于后续统一回测与迭代。

## 4. 入场条件（仅 3 个核心条件）
同时满足以下 3 项即触发 `enter_long`：
1. 多均线趋势成立：`ema_fast > ema_mid > ema_slow`
2. RSI 动量成立：`rsi > 55`
3. 成交量确认：`volume > volume_ma`

说明：仅额外保留基础数据清洗 `volume > 0`，不引入额外复杂过滤。

## 5. 出场条件
- `populate_exit_trend()` 保持极简，不增加复杂技术离场；
- 主要由以下全局参数控制：
  - `minimal_roi = {"0": 1.0}`
  - `stoploss = -0.99`

## 6. 风险点
1. 20x 杠杆会放大净值波动，短时回撤体感非常明显；
2. 固定 20 USDT 虽然限制了单笔资金暴露，但连续亏损仍会累积；
3. 止损非常宽（-99%）意味着策略对单笔浮亏容忍极高，需接受高风险实验属性；
4. 入场过滤较少，在噪音行情中可能出现更多无效信号。

## 7. 最适合的行情
- 中短周期内持续性上行、回调不深、成交量配合良好的趋势行情。

## 8. 最容易失效的行情
- 高频震荡、假突破频发、量价背离明显的横盘或弱趋势行情。

## 9. 为什么这是“简单趋势跟随 + 20x + 20USDT 固定仓位”实验原型
1. 结构上仅保留 RSI / EMA / 成交量均线三类基础指标；
2. 入场核心逻辑压缩为 3 条，避免复杂链式条件导致过拟合；
3. 仓位与杠杆固定化：`custom_stake_amount()` 固定 20.0、`leverage()` 固定 20x（受 `max_leverage` 限制）；
4. 离场不做复杂主观干预，主要由统一 ROI/止损框架驱动，便于后续系统化评估。
